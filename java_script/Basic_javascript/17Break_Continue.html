Break & Continue

In JavaScript, "break" and "continue" are keywords used within loops to control the flow of execution.

Break Statement

break is used to immediately terminate the enclosing loop (for loop, while loop, or switch statement) and resume execution at the next statement after the loop.

for (let i = 0; i < 5; i++) {   if (i === 3) {    break; // Terminate the loop when i is equal to 3   }   console.log(i); }

Output : 0, 1, 2

Continue Statement

continue is used to skip the rest of the current iteration and proceed to the next iteration of the loop.

for (let i = 0; i < 5; i++) {   if (i === 2) {    continue; // Skip the iteration when i is equal to 2   }   console.log(i); }

Output : 0, 1, 3, 4
