JavaScript if...else Statement

In JavaScript, there are three forms of the if...else statement.

  1. if statement
  2. if...else statement
  3. if...else if...else statement

If statement

if Statement

It evaluates the content only if expression is true.

The syntax of the if statement is:

if (condition) {
   // the body of if
}

If...else Statement

If...else Statement

It evaluates the content whether condition is true of false.

The syntax of the If...else statement is:

if (condition) {
   // block of code if condition is true
} else {
   // block of code if condition is false
}

if...else if...else Statement

if...else if...else Statement

To check multiple conditions and execute the corresponding block if a condition is true, you use the if...else if...else statement

The syntax of the if...else if...else statement is:

if (condition1) {
   // code block 1
} else if (condition2){
   // code block 2
} else {
   // code block 3
}
