String

A string data type is a group of characters or textual content enclosed by single quotes (' ') or double-quotes (" "), or tick signs (` `).

Number

A number data type is a numeric value. It can be an integer (positive or negative), a floating-point value, or an exponential value.

Boolean

Boolean is a logical data type that is used to compare two variables or to check a condition. It has only two values, namely, true and false.

Undefined

Undefined data type means that a variable is just declared and is not assigned any value. It is the default value of a variable that has not been defined.

Null

A null value means no value. It means an empty value or absence of value.

Object

An object data type stores multiple values or a collection of data in terms of properties and methods. It contains key-value pairs in its address. The properties of the object are written as key:value pairs. Each pair is separated by commas, enclosed in curly braces {}. The key must be a string and the value can be of any data type.
