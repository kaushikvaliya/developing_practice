JavaScript switch case

The JavaScript switch statement is used for multi-way branching. It evaluates an expression and matches it against multiple cases, executing the code block associated with the matching case.

The syntax of the switch case statement is:

switch(variable/expression) {
case value1:
   // statement1;
break;
case value2:
   // statement2;
break;
case value3:
   // statement3;
break;
default:
   // statement
}
