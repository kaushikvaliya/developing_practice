ES6 Dialog boxes

There are three types of dialog boxes supported in JavaScript that are alert, confirm, and prompt. These dialog boxes can be used to perform specific tasks such as raise an alert, to get confirmation of an event or an input, and to get input from the user.

Alert Dialog box

alert : Displays a simple message box with an OK button. It is used to show information to the user.

The syntax of the alert box is:

alert(message);

Confirmation Dialog box

confirm : Displays a dialog box with a message and OK and Cancel buttons. It is used to get a binary choice from the user (OK or Cancel).

The syntax of the Confirmation box is:

confirm(message);

Prompt Dialog box

prompt(message, defaultText): Displays a dialog box with a message, an input field for the user to enter text, and OK and Cancel buttons. It is used to get input from the user.

The syntax of the Confirmation box is:

Prompt(message);
